# 🚀 クイックスタートガイド

## 最も簡単な起動方法

### ステップ1: app.py を実行

フォルダ内でコマンドプロンプトを開き、以下を実行してください。

```cmd
python app.py
```

### ステップ2: 初回起動時の設定

初回起動時のみ、以下の手順が必要です:

1. **APIキーの入力を求められます**
   ```
   APIキーを入力しますか? (y/n): y
   ```
   → `y` を入力してEnter

2. **APIキーを貼り付けます**
   ```
   APIキーを入力してください (sk-ant-...):
   ```
   → あなたのAnthropicのAPIキーを貼り付けてEnter

3. **自動的に保存されます**
   ```
   ✓ APIキーを config.txt に保存しました
   ```

### ステップ3: 使用開始

1. ブラウザが自動的に開きます
2. 記事のURLまたはテキストを入力
3. 「図解を生成」ボタンをクリック（生成中はローディングが表示されます）
4. 生成完了後に表示される「図解を開く」ボタンをクリックすると、既定のブラウザで図解が開きます
5. 完了！
   図解は `output/` フォルダに保存されます

---

## APIキーの取得方法

Anthropicのアカウントを持っていない場合:

1. https://console.anthropic.com/ にアクセス
2. アカウント作成（Googleアカウントでもサインアップ可能）
3. **Settings** → **API Keys** に移動
4. **Create Key** をクリック
5. 生成されたキー（`sk-ant-api03-...`）をコピー

## トラブルシューティング

### Q: 「ModuleNotFoundError」が出る

A: `app.py` が自動でインストールします。
インターネット接続を確認してください。

手動でインストールする場合:
```cmd
pip install anthropic requests beautifulsoup4
```

### Q: 「Failed to fetch」エラーが出る

A: サーバーが起動していません。
以下を確認:
1. `python app.py` が実行されているか
2. `http://localhost:8000` にアクセスできるか
3. ファイアウォールがブロックしていないか

### Q: 図解が生成されない

A: 以下を確認:
1. APIキーが正しいか（`config.txt` を確認）
2. Anthropic APIの残高があるか
3. インターネット接続があるか

## 次回以降の起動

次回からも `python app.py` を実行するだけです！
APIキーは自動的に読み込まれます。

---

## 終了方法

次のいずれかで終了できます。

- ブラウザ画面の「サーバーを停止」ボタンをクリック（確認ダイアログで「OK」を選択）。
  停止後に「このウィンドウを閉じてください」と表示されるので、ブラウザのタブを閉じてください。
- コマンドプロンプトのウィンドウで `Ctrl+C` を押す
- コマンドプロンプトのウィンドウを閉じる

再度使うときは、もう一度 `python app.py` を実行してください。
